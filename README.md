# William Qi Portfolio

React + Vite portfolio site for `https://willsqqi.github.io/`.

## Local Development

```bash
npm install --cache ./.npm-cache
npm run dev
```

## Verification

```bash
npm test
npm run build
```

## Deployment

Push this repository to `willsqqi/willsqqi.github.io` and enable GitHub Pages with GitHub Actions as the source. The included workflow builds and deploys the site on pushes to `main`.
